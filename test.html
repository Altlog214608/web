      #html 태그 : html 문서 root 구역
      #head 태그 : 문서의 제목, 문서에 대한 설명, 메타 태그 등 설명
      #body 태그 : 문서의 실질적인 본문 구역
      #title 태그 : 페이지의 타이틀 설정 부분( 브라우저의 타이틀 바 ) head 태그의 자식으로 넣어야함
      #h태그 : 제목 태그 h1부터 h6까지 사용, h는 head를 의미
      #p태그 : 단락 나누기, 본문 문단 구성하는 태그
      #hr태그 : 수평선 생성, 종료태그 없음
      #br태그 : 새로운 줄로 넘어가기, 종료태그 없음음
      #strong 태그 : 중요한 내용
      #em 태그 : 강조 표현
      #i 태그 : 이탤릭체
      #del 태그 : 삭제표기
      #ins 태그 : 추가표기
      #sup 태그 : 윗첨자
      #sub 태그 : 아래첨자

      #small 태그 :작은 문자

      #div 태그와 span 태그는 영역 설정 역할
      #div 태그 : 블록을 구성하는 HTML에서 가장 많이 쓰이는 태그
      div태그는 특별한 의미를 지닌 것은 아니라 특정 대상들을 블록 구조로 묶어
      컨테이너로서의 역할을 함
      -블록태그형테 -> 컨텐츠 양과 무관하게 웹 좌측 끝부터 우측 끝까지 영역을 차지하는 형태

      #span 태그 : span 태그는 텍스트 일부분에 영역을 지정할 때 사용
      -인라인 태그 -> 태그로 감싼 부부분만 영역이 해당되는 태그
      
      태그 구분 기준 1.종료태그의 유무 2.인라인과 블럭럭

      #iframe 태그
      iframe태그를 이용하면 현재 페이지 내 내장 윈도우(창)을 하나 만들고
      다른 HTML 페이지를 출력할 수 있다. iframe을 통해
      만들어지는 새 윈도우(프레임)를 인라인 프레임이라고 함.

      #audio와 video
      HTML5 이전에는 플러그인 설치가 필요했다
      플러그인: 비규격(비표준)형식의 데이터를 다루기 위한 부가 설치 항목
      HTML5 이후 표준화되어 설치 필요X
      오디오 : audio 태그
      비디오 : video 태그
      규격화 되지 않은 매체: embed태그 혹은 object태그로 작성성

      mp3 : mp3로 인코딩된 유형
      wav : wave 형식 인코딩 유형

      #웹 폼
      #웹 폼이란 사용자를 통해 입력을 받는 형태
      input 태그 / textarea 태그 / select 태그 등을 사용한다.
      input 태그는 type을 어떻게 지정하냐에 따라 버튼 or 입력창 외
      다양한 형태로 생성된다.

      form 태그의 name 속성은 JS에서 폼을 지정할 때 식별하는 이름
      form 태그의 action 속성은 폼데이터를 처리할 웹서버 응용 프로그램 지정하는 속성
      form 태그의 method 속성은 폼 데이터를 웹 서버로 전송하는 형식
      지정하는 속성, 대표적으로 GET과 POST가 있음
