CSS 스타일 시트 개요

CSS 적용 타겟 샘플문장입니다

Selector는 CSS 스타일 시트의 이름 규칙, 스타일 시트의 블록은 중괄호로 처리한다. css스타일 프로퍼티와 값을 쌍으로 설정한다. CSS는 200개 정도 프로퍼티(속성)이 있음 계속 추가되는 중

CSS의 주석처리

CSS의 주석은 /*~~~*/ 로 주석 가능

CSS속성은 대소문자를 구분하지 않음

CSS스타일 시트를 적용하는 방법 3가지

스타일 적용 방법 1 : head 내에 style 태그로 구성하는 방법

스타일 적용 방법 2 : 태그의 style 속성에 직접 적용 하는 방법

스타일 적용 방법 3 : 외부 스타일 시트를 import 혹은 link로 불러와서 사용하는 방법

스타일 태그를 통해 스타일 시트 만들때는 head태그 내에 style 태그를 작성한다

CSS3스타일 시트만 떼어내어 .css 확장자를 가진 파일을 만들어 스타일 시트를 넣어 저장하고 필요할 때 불러와서 사용할 수 있음

link태그를 이용한 외부 스타일 시트 적용

link 의 href 속성에 적용하려는 외부 스타일 시트 파일의 경로, 파일 명을 작성한다

CSS의 규칙

CSS스타일 시트에는 규칙이 있다.

1.css스타일은 부모태그로부터 상속된다 자신을 둘러싸는 태그를 부모태그 혹은 부모 요소 (element)라고 하는데, 부모 태그에 적용된 스타일은 자식 요소에 상속 적용 됨.

스타일 합치기와 오버라이딩

브라우저는 각 태그에 대한 디폴트 스타일을 가지고 있음 따라서 개발자가 따로 지정하는 스타일이 없다면 브라우저 기본 디폴트 값을 통한 스타일이 적용 됨. 예) padding, margin 은 0으로 초기화 직접 해줌

특정 태그에는 4가지 종류의 스타일 시트가 적용될 수 있음

1.브라우저 디폴트 2. 외부CSS파일 시트 3.내부 스타일 시트 4.인라인 스타일 시트

스타일 합치기란 4가지 스타일 시트가 태그에 동시 적용될 때 스타일이 합쳐져 적용됨을 의미

스타일 오버라이딩이란 덮어쓰기의 의미로 4가지 루트 중 같은 속성(예를들어 폰트크기) 가 여러번 적용되면 우선순위가 높은 스타일시트로 덮어씌워 짐

우선순위 1번부터 4번으로 갈수록 높아짐
